## 选中多行添加注释
```go
// 先按 v 键进入 visual mode，然后输入 :s/^/\/\/ / 就添加了注释 "// "
:s/    表示开始一个替换命令
^/     正则匹配行首
\/     插入一个斜杠（因为 / 在命令里是特殊字符，要转义）


// :s/^\/\/ / 取消注释 "// "



// 使用 lua 配置，<leader>df 添加注释 "// "，<leader>re 删除注释
vim.keymap.set({'n', 'v'}, '<leader>df', ':s/^/\\/\\/ /<CR>')
vim.keymap.set({'n', 'v'}, '<leader>re', ':s/^\\/\\/ //<CR>')
```


```
:e index.html 打开其他文件

gd 调整到变量定义的位置
gk 跳转回来

<leader>lf 格式化文件
<leader>uf 打开/关闭文件树

<leader>bl 下一个窗口
<leader>bh 关闭当前窗口

s 然后输入两个字母再输入代码中红色的字母          跳转到指定单词位置

<leader>df 为代码添加 "// "注释
<leader>re 删除 "// " 注释





d3w 删除3个单词
d3W 删除3个连续字符串
w 移动到下一个单词或标点符号
e 移动到单词末尾
b 移动到单词开头
ge 移动到上一个单词末尾


r 然后输入一个字母                替换当前字母为输入的字母
R 进入替换模式

u 撤销
Ctrl + r 重做redo
```
