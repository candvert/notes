## 日语输入法
```sh
按两个n打出ん
促音っ按下一个假名的首字母的两次或xtsu打出     比如北海道/ほっかいどう需要按下hokkaidou
假名注音前加x打出小写   比如ディ需要按下dexi
ヴ   vu
英日切换Shift + CapsLock
F7切换为片假名
F6切换为平假名
```
## 发音
```sh
“た”行的“た ta” “て te” “と to”的辅音与汉语“t” “d”的发音相当。出现在词头时，与“t”相近，出现在词中时，与“d”相近。基于同样的道理，“ち”出现在词头和词中时，分别相当于汉语的“qi” “ji”，“つ”出现在词头和词中时，分别相当于汉语的“ci” “zi”。


外来词专用音节：
ツァ  tsa      ファ  fa
ウィ　wi       ティ  ti      フィ  fi      ディ  di
トゥ　tu       ドゥ  du      デュ  dyu
ウェ　we       シェ  she     チェ  che     ツェ  tse     フェ  fe    ジェ  je
ウォ　wo       ツォ  tso     フォ  fo
```

## 动词
```sh

动词的“基本形”　“ない形”　“た形”都属于简体形。


て形表示动作相继发生、正在做某事、请求某人做某事、许可、禁止、并列、动作结束后留下的结果状态、反复、习惯
```

## 一些基础语法知识
```sh
助词有多个种类：
    附于名词后，表示与谓语的关系等：が，を，に，へ，から，と，で
    表示小句与小句之间的接续关系：から，ので
    添加特别的意义：は，も，こそ，さえ，だけ
    位于句尾表示语气：か，よ，わ，ね


“ます形”的“ます”具有与动词同样的词形变化，可以分别变为“書きまして（て形）”　“書きました（た形）”　“書きましょう（意志形）”等形式。
“ない形”的“ない”具有与形容词同样的词形变化，可以分别变为“書かなくて（て形）”　“書かなかった（た形）”　“書かなければ（ば形）”等形式。


一类形容词、二类形容词、名词做谓语的否定形式，都是将“て形”去掉“て”后续形容词“ない”。但是敬体形式的否定形式有“～ないです”和“～ありません”两种。


名词修饰名词时使用“名词+の+名词”的形式；动词、一类形容词修饰名词时直接后续名词；二类形容词修饰名词时使用“二类形容词+な+名词”的形式。

表示过去的意义、性质已发生改变等情形，可以使用过去形式修饰名词。
子供のころにおもしろかった映画が，今はもうおもしろくない。



二类形容词和名词做谓语，除了使用“～だ”　“～です”的形式以外，还可以使用“～である”　“～であります”的形式。就是在“て形”后面添加“ある”或“あります”。



“では”在口语里一般说成“じゃ”。



“だれ”的比较礼貌的说法是“どなた”。



これ/この/ここ/こちら


“一百”“一千”在日语中是“百”“千”，前面不加“1”，但“一万”“一千万”“一亿”则是“1万”“1千万”“1億”，前面要加“1”。



“いいです”的否定形式是“よく　ないです”或“よく　ありません”。
“いいです”的过去形式是“よかったです”，过去否定形式是“よく　ありませんでした”。
“多い”　“少ない”不同于其他一类形容词，不能单独修饰名词。



三类动词“します”前接汉字词或外来词可以构成新的三类动词，如“掃除します”。以这种形式构成的三类动词也可以说成“名词+を+します”，如“勉強を　します”。



表示人或物体的附着点，即人或物体停留在交通工具或椅子上等时，附着点用助词“に”表示。这种情况绝对不能用“を”表示。



“何”　“だれ”　“どこ”　“いつ”等表示疑问的词做主语时不能用“は”，只能用“が”。因为这些词不能表示旧信息。


疑问词“いつ”　“どこ”　“だれ”等词后续“か”，表示“某时”　“某地”　“某人”等意思。
“いつか”　“どこか”　“だれか”　“何か”等本身并不表示疑问的意思，所以可用于陈述句。


简体的会话中常常省略助词。


“ありません”的简体形是“ない”。


新信息做主语时用“が”来表示。


～って表示传闻。



助词“は”用于提示主题。“は”还可以表示对比。
用“を”前面的名词作话题，或者进行对比时，可替换为“は”。
助词“か”接在句尾表示疑问。
存在的场所用助词“に”表示。
表示动作发生的时间时，要在具体时间词语后面加上助词“に”。
“会います”所见到的对象用助词“に”表示。
表示移动行为的目的时用助词“に”。
表示人或物体的附着点，即人或物体停留在交通工具或椅子上等时，附着点用助词“に”表示。这种情况绝对不能用“を”表示。
“に”也可以用来表示用途和基准。表示用途时，其前面是具体说明用途的名词，后面一般是“使います”等动词。表示基准时，其前面的名词是基准，后面则一般是表示评价的形容词。
移动行为的目的地既可以用“へ”表示，也可以用“に”表示。
移动行为的目的地用助词“へ”表示。
交通手段用助词“で”表示。不使用交通工具而步行时用“歩いて”。
动作的场所用助词“で”表示。
“で”可以用来表示其他手段以及原材料。
某事发生的场所用“で”表示。
助词“で”加在名词后面还可以表示原因、理由。
动作的对象用助词“を”表示。
经过的场所用助词“を”表示。
当自动词表示移动行为时，移动的“经过点”和“起点”用助词“を”表示，这里的“を”不表示宾语。
自动词其主语用“が”表示。
日本人常常不把话说完，用接续的助词“が”来收尾。
表示两个小句的转折关系时，用助词“が”。
“が”还可以用于铺垫。
新信息做主语时用“が”来表示。
叙述直接观察到的某事物，而不是作为话题在谈论这个事物时，使用“[名词]が　[动词]”的形式。因为这里也同样是新信息在做主语。
“何”　“だれ”　“どこ”　“いつ”等表示疑问的词做主语时不能用“は”，只能用“が”。因为这些词不能表示旧信息。
“～か　どうか”小句中的主语后面必须用“が”，而不能用“は”。
修饰名词的动词小句中的主语，不用“は”而用“が”来表示。
[名]が　[名]です
[名]が　[一类形]/[二类形]です

对若干名词进行选择时，名词和名词之间加“か”，相当于汉语的“或者”。
“や”用于列举许多项目中的两项。
助词“と”加在两个名词之间表示并列，意思相当于汉语的“和”。
共同做某事的对象用助词“と”表示。


“に”的用法有：表示存在的场所，如“公園に　ベンチが　あります”；表示时间，如“5時に　帰ります”；表示动作的对象，如“友達に　会います”；表示移动行为的目的，如“本を　買いに　行きます”；表示附着点，如“車に　乗ります”。这些用法不能用“へ”来替换，只有表示“行きます”　“来ます”　“帰ります”等典型的移动动词的目的地时，“に”和“へ”才可以通用。


全面否定“名词+を+动词”时，使用“疑问词+も+动词否定形式”。全面否定“名词+に/から/と+动词”时，在这些助词后面加“も”成“疑问词+にも/からも/とも+动词否定形式”。全面否定“名词+へ+动词”时，既可以使用“疑问词+も+动词否定形式”，也可以使用“疑问词+へも+动词否定形式”。

————何も　したく　ありません。

————だれにも　会いたくないです。
どこへ　行きたいですか。
————どこへも　行きたくないです。

“あります”表示在某处发生某事的用法。
```

## 语法
```sh
第1课：
1. [名]は　[名]です
   李さんは　中国人です。
2. [名]は　[名]では　ありません
   “では”在口语中有时会发成“じゃ”。
   森さんは　学生では　ありません。
3. （1）[名]は　[名]ですか
   日语的问句在句尾不使用“？”。
   あなたは　小野さんですか。
   （2）应答
   森さんは　学生ですか。
   ————はい，そうです。
   ————いいえ，ちがいます。
   ————分かりません。
4. [名]の　[名]    （表示前面的名词是后面名词从属的机构、国家或属性。）
   李さんは　JC企画の　社員です。
第2课：
1. [これ/それ/あれ]は　[名]です
   これは　本です。
2. だれですか/何ですか
   それは　何ですか。
   あの人は　だれですか。
   “だれ”的比较礼貌的说法是“どなた”。
3. [名]の　[名]    （所属）
   わたしの　かぎ
4. [この/その/あの]　[名]は　[名]です
   この　カメラは　スミスさんのです。
5. どれ/どの　[名]
   森さんの　かばんは　どれ　ですか。
   小野さんの　机は　どの　机ですか。　
第3课：
1. [ここ/そこ/あそこ]は　[名]です
   ここは　デパートです。
2. [名]は　[名（场所）]です
   食堂は　デパートの7階です。
   小野さんは　事務所です。
3. [名]は　どこですか
   トイレは　どこですか。
4. [名]も　[名]です
   あそこも　JC企画のビルです。
5. [名]は　[名]ですか，[名]ですか
   かばん売り場は　1階ですか，2階ですか。
6. [名]は　いくらですか
   これは　いくらですか。
第4课：
1. “あります”用于不具有意志的事物，“います”用于具有意志的人、动物或昆虫。
   [名（场所）]に　[名（物/人）]が　あります/います
   部屋に　机が　あります。
   [名（物/人）]は　[名（场所）]に　あります/います
   いすは　部屋に　あります。
   小野さんの家は　どこに　ありますか。
2. [名]と　[名]    （并列）
   居間に　テレビと　ビデオが　あります。
3. 上/下/前/後ろ/隣/中/外
   表示具体位置时，用“名词+の+上/下/前/後ろ/隣/中/外”。
   机の　上に　猫が　います。
4. ね    （确认）
   当说话人就某事征求听话人的同意时，句尾用助词“ね”，读升调。
   あそこに　犬が　いますね。
5. 疑问词 + も + [动（否定）]
   表示全面否定。
   教室に　だれも　いません。
第5课：
1. 今　～時　～分です
   双方都明确在讲现在的时间时，“今”可以省略。询问具体时间时用“何時”。
   今　4時　です。
   今　何時　ですか。
   ————8時30分です。
   “30分”可以用“半”替代，有时表示具体时间的词前面还可以加上“午前”或“午後”。
   今　午前　8時半です。
2. [动]ます　/[动]ません
   [动]ました/[动]ませんでした
   森さんは　毎日　働きます。
3. [名（时间）]に　[动]
   表示动作发生的时间时，要在具体时间词语后面加上助词“に”，如“3時に”　“9時に”。
   森さんは　7時に　起きます。
   叙述包含数字的时间时后续助词“に”，如“3月14日に”　“2008年に”，但“今”　“昨日”　“今日”　“明日”　“毎日”　“去年”　“来年”等词后不能加“に”。星期的后面一般加“に”，如“日曜日に”，但也可以不加。
4. [名（时间）]から　[名（时间）]まで　[动]
   わたしは　9時から　5時まで　働きます。
5. いつ　[动]ますか
   試験は　いつ　始まりますか。
   询问的时间很具体时，在表示时间的词语后面加“に”，如“何時に”　“何曜日に”　“何日に”。
   仕事は　何時に　終わりますか。
6. は    （对比）
   小野さんは　今日は　休みます。
第6课：
1. [名（场所）]へ　[动]
   吉田さんは　中国へ　行きます。
2. [名（场所）]から　[动]
   李さんは　先月　北京から　来ました。
3. [名（人）]と　[动]
   小野さんは　友達と　帰りました。
4. [名（交通工具）]で　[动]
   上海まで　飛行機で　行きます。
   交通手段用助词“で”表示。不使用交通工具而步行时用“歩いて”。
   李さんは　歩いて　アパートへ　帰りました。
5. [名（场所）]から　[名（场所）]まで　[动]
   森さんは　東京から　広島まで　新幹線で　行きます。
6. “に/で/へ/から/まで/と” + は
   助词“は”表示对比时，可以单独使用也可以加在“に/で/へ/から/まで/と”的后面构成一种复合形式。
   わたしの　部屋には　電話が　ありません。
第7课：
1. [名]を　[动]
   李さんは　毎日　コーヒーを　飲みます。
2. [名（场所）]で　[动]
   李さんは　図書館で　勉強します。
3. [名]か　[名]
   对若干名词进行选择时，名词和名词之间加“か”，相当于汉语的“或者”。
   わたしは　毎朝　パンか　お粥を　食べます。
4. [名]を　ください
   コーラと　ケーキを　ください。
第8课：
1. [名（工具）]で　[动]
   李さんは　日本語で　手紙を　書きます。
2. [名（人）]は　[名（人）]に　[名（物）]を　あげます
   わたしは　小野さんに　お土産を　あげます。
3. [名（人）]は　[名（人）]に　[名（物）]を　もらいます
   わたしは　小野さんに　辞書を　もらいました。
   わたしは　長島さんから　写真を　もらいました。
4. [名（人）]に　会います
   李さんは　明日　長島さんに　会います。
5. よ    （提醒）
   すみません，李さんは　いますか。
   ————もう　帰りましたよ。
6. もう
   昼ご飯を　食べましたか。
   ————ええ，もう　食べました。
第9课：
1. [名]は　[一类形]です
   四川料理は　辛いです。
2. 一类形容词做谓语时的否定形式是将词尾的“い”变成“く”再加上“ないです”或“ありません”。
   このスープは　熱く　ないです。
   “いいです”的否定形式是“よく　ないです”或“よく　ありません”。
3. 一类形容词做谓语时的过去形式是将词尾的“い”变成“かった”再加“です”。其过去形式的否定形式则是把词尾“い”变成“く　なかったです”或者“く　ありませんでした”。
   旅行は　楽し　かったです。
   昨日は　寒く　なかったです。
   “いいです”的过去形式是“よかったです”，过去否定形式是“よく　ありませんでした”。
4. [一类形] + [名]
   広い　国
5. を -> は
   用“を”前面的名词作话题，或者进行对比时，可替换为“は”。
   わたしは　コーヒーは　飲みません。
6. あまり　[一类形]/[动]（否定）
   このスープは　あまり　熱く　ないです。
7. “とても”　“たいへん”　“少し”　“ちょっと”　“あまり”　“全然”等都是表示程度的副词。“あまり”　“全然”要与后面的否定形式呼应使用。
   この料理は　とても　おいしいです。
   試験は　全然　難しく　ありませんでした。
第10课：
1. [名]は　[二类形]です/でした
   京都の　紅葉は　有名です。
2. [名]は　[二类形]では　ありません/では　ありませんでした
   二类形容词做谓语时的否定形式是“二类形容词+では　ありません”。过去形式的否定形式是“では　ありませんでした”。在口语里“では”一般说出“じゃ”。
   この通りは　にぎやかでは　ありません。
3. [名]は　[二类形]な　[名]です
   奈良は　静かな　町です。
4. [名]でした
   名词做谓语时的过去形式是“名词+でした”。其否定形式是“名词+では（じゃ）　ありませんでした”。
   昨日は　日曜日でした。
   昨日は　雨では　ありませんでした。
5. どんな　[名]
   横浜は　どんな　町ですか。
6. どうですか/どうでしたか
   この料理は　どうですか。
   京都は　どうでしたか。
7. “でも” 和 “そして”
   わたしの部屋は　狭いです。でも，きれいです。
   わたしの部屋は　狭いです。そして，汚いです。
第11课：
1. [名]は　[名]が　[一类形]/[二类形]です
   小野さんは　歌が　好きです。
2. [名]は　[名]が　分かります/できます
   スミスさんは　韓国語が　分かります。
   小野さんは　料理が　上手です。
3. [名]や　[名]
   吉田さんは　時々　中国や　韓国へ　行きます。
   “や”常常和“など”呼应使用，构成“～や　～など”。
   わたしは　お店で　野菜や　果物などを　買いました。
4. “～から” 和 “だから”    （原因、理由）
   森さんは　お酒が　好きですから，毎日　飲みます。
   キムさんは　野菜が　嫌いです。だから，あまり　野菜を　食べません。
   “だから”的比较礼貌的说法是“ですから”。
   明日は　休みです。ですから，子供と　動物園へ　行きます。
5. “いつも”　“よく”　“時々”　“たまに”　“あまり”　“全然”等是表示频率的副词。“あまり”　“全然”后面须接否定形式。
   わたしは　たまに　コンサートへ　行きます。
   田中さんは　あまり　横浜へは　行きません。
6. どうしてですか
   回答时在谓语后面加“から”。
   甲：ぼくは　来月　また　ここへ　来ますよ。
   乙：どうしてですか。
   甲：ここで　友達の結婚式が　ありますから。
第12课：
1. [名]は　[名]より　[一类形]/[二类形]です
   李さんは　森さんより　若いです。
2. [名]より　[名]の　ほうが　[一类形]/[二类形]です
   日本より　中国の　ほうが　広いです。
3. [名]は　[名]ほど　[一类形]く　ないです/[二类形]です　ありません
   東京の冬は　北京の冬ほど　寒く　ないです。
4. [名]の　中で　[名]が　いちばん　[一类形]/[二类形]です
   スポーツの　中で　サッカーが　いちばん　おもしろいです。
   [名（场所/时间）]で　いちばん　[一类形]/[二类形]　[名]は　[名]です
   日本で　いちばん　高い　山は　富士山です。
5. [名]と　[名]と　どちらが　[一类形]/[二类形]ですか
   回答时使用“～の　ほうが···”。如果两者程度相当，则使用“どちらも　～です”。“どちら”在口语中有时说成“どっち”。
   日本語と　英語と　どちらが　難しいですか。
   ————日本語の　ほうが　難しいです。
6. どの　[名]/いつ/どれ/だれ/何が　いちばん　[一类形]/[二类形]ですか
   この中で　どの　料理が　いちばん　おいしいですか。
第13课：
1. [名（数量）] + [动]
   卵を　1個　食べます。
2. [名（时间）] + [动]
   表示动作、状态的持续时间，这时候表示时间数量的词语后面不能加“に”。
   李さんは　毎日　7時間　働きます。
3. [名（时间）]に　[名（次数）] + [动]
   李さんは　1週間に　2回　プールへ　行きます。
4. 动词的“ます形”
5. [名（场所）]へ　[动]に　行きます/来ます
   动词用“ます形”去掉“ます”的形式。
   午後　郵便局へ　荷物を　出しに　行きます。
6. [名（数量）] + で
   このケーキは　3個で　500円です。
   数量为1个时，不加“で”。
   このケーキは　1個　200円です。
第14课：
1. 动词的“て形”
2. [动]て　[动]    （相继发生）
   昨日　デパートへ　行って，買い物しました。
3. [动]てから　[动]    （相继发生）
   “てから”不能在一个句子中反复使用两次以上。
   李さんは　毎晩　ラジオを　聞いてから　寝ます。
4. [动]て　ください
   请求某人做某事。
   ここに　住所と　名前を　書いて　ください。
5. [名（场所）]を　[动]    （经过/离开）
   经过的场所用助词“を”表示。
   このバスは　駅前を　通ります。
   李さんは　毎朝7時に　家を　出ます。
第15课：
1. [动]て　います    （正在进行）
   表示动作或变化正在进行。
   小野さんは　今　新聞を　読んで　います。
2. [动]ても　いいです
   家で　仕事を　しても　いいです。
3. [动]ては　いけません
   飛行機の　中で　タバコを　吸っては　いけません。
   对尊长一般不用“～ては　いけません”而使用“～ないで　ください”。
4. [名（附着点）]に　[动]
   附着点用助词“に”表示。
   小野さんは　公園で　ボートに　乗りました。
5. [名（目的地）]に　[动]
   移动行为的目的地既可以用“へ”表示，也可以用“に”表示。
   李さんは　病院に　行きます。
第16课：
1. [一类形]て　[一类形]    （并列）
   ホテルの部屋は　広くて　明るいです。
   軽くて　暖かい　コートを　買いました。
2. [二类形]で　[二类形]    （并列）
   このコンピュータの操作は　簡単で　便利です。
   一类形容词和二类形容词也可以连接使用。
   この公園は　静かで　広いです。
3. [名]で　[名]    （并列）
   スミスさんは　旅行会社の社員で，営業部の部長です。
4. [动]て　います    （结果状态）
   森さんは　車を　持って　います。
   对用表示结果状态的“～て　います”提问的句子做否定回答时，使用“～て　いません”。但是，对“知って　いますか”做否定回答时必须使用“知りません”。
   田中さんを　知って　いますか。
   ————いいえ，知りません。
5. [小句]が，[小句]    （转折）
   3か月　パソコンを　勉強しましたが，まだ　あまり　できません。
6. まだ + [动（否定）]
   何年ぐらい　日本語を　勉強しましたか。
   ————2年　勉強しました。でも，まだ　あまり　できません。
   对用“もう”提问的句子做否定回答时，使用“まだ　～て　いません”的表达形式。
   小野さんは　もう　結婚して　いますか。
   ————いいえ，まだ　結婚して　いません。
第17课：
1. [名]が　欲しいです
   わたしは　新しい　洋服が　欲しいです。
2. [名]を　[动]たいです
   “たい”前接动词“ます形”去掉“ます”的形式。
   わたしは　映画を　見たいです。
   “たい”的对象有时不用“を”，而用“が”表示。但“欲しい”的对象只能用“が”来表示。
   水が　飲みたいです。
3. [动]ませんか
   いっしょに　お茶を　飲みませんか。
4. [动]ましょう
   把动词“ます形”的“ます”换成“ましょう”，可表示提议。
   ちょっと　休みましょう。
5. 疑问词 + でも
   李さん，何を　食べたいですか。
   ————何でも　いいです。
6. ね    （缓和语气）
   日本料理で　何が　いちばん　好きですか。
   ————そうですね。やっぱり　お寿司ですね。
第18课：
1. [一类形]　なります
   表示性质或状态的变化。构成方式是把一类形容词的词尾“い”变成“く”，再加“なります”。
   だんだん　暖かく　なります。
2. [一类形]　します
   用于因主语的意志性动作、作用而引起事物发生变化的场合。构成方式是把一类形容词的词尾“い”变成“く”，再加“します”。
   テレビの音を　大きく　します。
3. [二类形]/[名]に　なります
   もう　元気に　なりました。
4. [二类形]/[名]に　します
   部屋を　きれいに　して　ください。
   在餐馆里回答服务员的点餐询问时，可以使用“～に　します”这一形式，例如：“カレーに　します”　“コーヒーに　します”。这是把自己决定的事情告诉对方的一种表达方式。
5. [一类形]/[二类形]　ほうが　いいです
   旅行の荷物は　軽い　ほうが　いいです。
   二类形容词使用“二类形容词+な+ほうが　いいです”的形式。
   公園は　静かな　ほうが　いいです。
6. 自动词和他动词
   自动词是不带宾语的动词，表示不考虑外力影响的情况下，主语自主地进行动作，或自然地发生变化，其主语用“が”表示。最典型的他动词具有主语作用于宾语而使其发生变化的含义，其宾语用“を”表示。
   当自动词表示移动行为时，移动的“经过点”和“起点”用助词“を”表示，这里的“を”不表示宾语。
   “～なります”是自动词，“～します”是他动词。
第19课：
1. 动词的“ない形”
2. [动]ないで　ください
   部屋のかぎを　忘れないで　ください
3. [动]なければ　なりません
   李さんは　今日　早く　帰らなければ　なりません。
   也可以用“ないと　いけません”的形式代替“なければ　なりません”。
   すぐに　書類を　送らないと　いけません。
4. [动]なくても　いいです
   明日は　残業しなくても　いいです。
5. [名]が　[名]です
   新信息做主语时用“が”来表示。
   山田先生は　どの方ですか。
   ————山田先生は　あの方です。
   ————あの方が　山田先生です。
   叙述直接观察到的某事物，而不是作为话题在谈论这个事物时，使用“[名词]が　[动词]”的形式。因为这里也同样是新信息在做主语。
   子供が　公園で　遊んで　います。
   バスが　来ました。
   “何”　“だれ”　“どこ”　“いつ”等表示疑问的词做主语时不能用“は”，只能用“が”。因为这些词不能表示旧信息。
   何が　好きですか。
第20课：
1. 动词的“基本形”
2. [名（人）]は　[动（基本型）]　ことが　できます
   スミスさんは　ピアノを　弾く　ことが　できます。
3. [名]は　[动（基本型）]　ことです
   当谓语部分的内容是一种动作、行为时使用这个句型。
   わたしの趣味は　切手を　集める　ことです。
   也可以用名词或名词短语来代替。
   小野さんは　車を　運転する　ことが　できません。
   小野さんは　車の　運転が　できません。
4. [动（基本型）]　前に，～
   こちらへ　来る　前に　電話を　かけて　ください。
   也可以以“名词+の+前に”的形式使用。
   会議の　前に，資料を　コピーします。
5. 疑问词 + か
   疑问词“いつ”　“どこ”　“だれ”等词后续“か”，表示“某时”　“某地”　“某人”等意思。
   部屋に　だれか　いますか。
   “いつか”　“どこか”　“だれか”　“何か”等本身并不表示疑问的意思，所以可用于陈述句。
   冬休み，どこかへ　行きたいです。
6. ～よね
   手作りの餃子の皮は　おいしいですよね。
第21课：
1. 动词的“た形”
2. [动（た型）]　ことが　あります
   北京へ　行った　ことが　ありますか。
   ————いいえ，一度も　行った　ことが　ありません。
   ————いいえ，ありません。
3. [动（た型）]　後で，～
   会社が　終わった　後で，飲みに　行きます。
   也可以以“名词+の+後で”的形式使用。
   仕事の後で，映画を　見ます。
4. [动（た型）]　ほうが　いいです
   もっと　野菜を　食べた　ほうが　いいですよ。
   否定表达形式为“动词ない形 + ほうが　いいです”。
   そんなに　慌てない　ほうが　いいですよ。
5. [动]ましょうか    （提议）
   何か　食べましょうか。
   回答有“お願いします”　“はい，そうしましょう”　“いいですね”，反对或拒绝时则说“～ないで　ください”或“いいえ，ちょっと・・・”。
   荷物を　持ちましょうか。
   ————はい，お願いします。
   窓を　閉めましょうか。
   ————いいえ，閉めないで　ください。
第22课：
1. 敬体形和简体形
   “～ます” “～です”属于敬体形，动词的“基本形” “ない形” “た形”都属于简体形。
   但一类形容词、二类形容词和名词本身没有“敬体形”和“简体形”之分，只有在做谓语时，其谓语形式才具有“敬体形”和“简体形”。
2. 动词的简体形
   森さんは　毎晩　テレビを　見る。
   简体的会话中常常省略助词。另外，“ありません”的简体形是“ない”。
   コーヒー，飲む？
3. 一类形容词谓语形式的简体形
   そのカレー，おいしい？
   ————ううん，あんまり　おいしく　ないよ。
4. 二类形容词谓语形式的简体形
   现在将来形式的肯定疑问句用去掉简体形中的“だ”的形式。
   このネクタイ，派手？
   ————ううん，全然　派手じゃ　ないわ。
5. 名词谓语形式的简体形
   现在将来形式的肯定疑问句用去掉简体形中的“だ”的形式。
   今日は　曇りだ。
6. [小句]けど，[小句]    （转折/铺垫）
   昨日の試験，どうだった？
   ————ちょっと　難しかったけど，まあまあ　できたよ。
   来週，送別会を　するけど，都合は　どうかな？
第23课：
1. [动]たり　[动]たり　します
   小野さんは　休みの日，散歩したり　買い物に　行ったり　します。
2. [一类形]かったり　[一类形]かったりです
   飛行機のチケットは　高かったり　安かったりです。
   [二类形]だったり　[二类形]だったりです
   この公園は　時間に　よって　静かだったり　にぎやかだったりです。
   [名]だったり　[名]だったりです
   日本語の先生は　中国人だったり　日本人だったりです。
3. （1）[小句] + か　どうか
   将不包含疑问词的疑问句作为一个长句中的一部分时，将动词、一类形容词的敬体形变为简体形，再加“か　どうか”。
   わたしは　今年の夏，北京へ　行くか　どうか　分かりません。
   也可以重复使用动词的“基本形”和“ない形”，表达的含义与上面的句子相同。
   わたしは　今年の夏，北京へ　行くか　行かないか　分かりません。
   “～か　どうか”直接接在名词或二类形容词后面，其间不加“だ”。“～か　どうか”小句中的主语后面必须用“が”，而不能用“は”。
   わたしは　キムさんが　中国人か　どうか　知りません。
   わたしは　キムさんが　中国人か　中国人では　ないか　知りません。
   （2）[疑问词小句] + か
   将包含“何”　“だれ”　“どこ”等的疑问句，作为一个长句中的一部分使用时，需将疑问词小句的动词、一类形容词的敬体形变为简体形，再加“か”。
   かぎが　どこに　あるか　教えて　ください。
   “～か”直接接在名词或二类形容词后面，其间不加“だ”
   この歌が　だれの歌か　知って　いますか。
第24课：
1. [小句（简体形）]と　思います
   李さんは　もう　すぐ　来ると　思います。
   “～と　思います”只能表示说话人的思考内容，但“～と　思いました”　“～と　思っています”既可以表示说话人的思考内容，也可以表示说话人以外的人的思考内容。
   馬さんは　日本へ　行きたいと　思って　います。
2. [名（人）]は　[小句（简体形）]と　言いました
   向第三者转述他人所说的话时，在所说的话后加“～と　言いました”。
   陳さんは　パーティーに　行くと　言いました。
   如果想明确指出是向谁说的，则在句子中加入“名词+に”。
   小野さんは　李さんに　ちょっと　休みたいと　言いました。
   如果转述某人反复说起的事情时，用“～と　言って　います”表示。
   来週のハイキングに　張さんも　行きたいと　言って　います。
   “～と　思います”及“～と　言いました”前面出现名词或二类形容词小句时必须加“だ”。
   あそこは　駅だと　思います。
3. （1）～のです/んです
   “～のです/んです”表示所讲的内容与前句或前项内容有关联。用于说明状况或解释原因、理由。
   すみません，頭が　痛いんです。
   “～のです/～んです”前接简体形，但名词和二类形容词的现在将来形的肯定形，“だ”要换成“な”。
   これ，京都のお土産なんです。どうぞ。
   （2）どうして　～のですか/んですか
   回答此问句时也要用“～のです/んです”的形式。
   どうして　食べないんですか。
   ————もう　おなかが　いっぱいなんです。
4. [小句]が，[小句]    （铺垫）
   東京タワーへ　行きたいんですが，どうやって　行きますか。
```

```sh
第1课：
1. 人称
   （1）“わたし”　“あなた”　“あの人”
   （2）～さん
   称呼小孩的时候，一般在其名字后加“ちゃん”。

   “お願いいたします”更加礼貌。对方回答时一般先说“こちらこそ”。
5. “会社員” 和 “社員”
   “会社員”表示一种职业，意思是“公司职员”。

第2课：
1. 方
   あの方は　田中さんです。
2. 询问年龄
   使用“何歳ですか”，或比较礼貌的“おいくつですか”。

3. どうぞ

   （1）えっ
   （2）わあ

   “ええ”是“はい”的比较随便的说法，但只用于肯定对方的提问。在别人叫到自己的名字而应答时，必须用“はい”而不能用“ええ”。
7. どうも　ありがとう　ございます
   在略示谢意时，可以只用“どうも”。
第3课：
1. 1階
   日语的量词和数词搭配使用时，有时候会有多种发音的情况。
2. 省略    （谓语的省略）
3. 礼貌语言
   （1）こちら/そちら/あちら/どちら
   “こちら/そちら/あちら/どちら”是“ここ/そこ/あそこ/どこ”的礼貌说法。
   （2）“お国は　どちらですか”    “会社は　どちらですか”
   “会社は　どちらですか”包含两种含义，一种是询问公司所在地，另一种是询问所属公司的名称。
4. 缩略词
   “パソコン”由“パーソナルコンピュータ”（英语的“personal computer”）缩略而来。这样形成的词叫缩略词。
5. あのう    （搭话）

   あのう，東京の地図は　どこですか。
   ————地図ですか（↗）。そちらです。
   それは　パソコンです。
   ————パソコンですか（↘）。
第4课：
1. “上”所表示的范围
   日语的“上”只表示垂直上方的范围。
   日语中墙壁的表面用“壁に”。
   壁に　スイッチが　あります。
2. ええと
3. “ご家族”　“ご兄弟”　“ご両親”
4. 兄弟
   兄妹、姐弟、姐妹也称“きょうだい”。
5. “JR” 和 “地下鉄”
   “JR”是“Japan Railways”的简称。
6. 一人暮らし
第5课：
1. 人称    （在工作单位）
   在公司里称呼上司时，一般可直接称呼其职务，像“課長”，或者在其职务前加上姓，如“吉田課長”。
   如果谈及对方单位的人，或者是与别的公司的科长或总经理交谈时，一般要在其职务后面再加上“さん”，以表示对对方的敬意。如“田中課長さん” “社長さん”。
2. “～です”的用法
   部分已经明确的内容，回答别人的问题不需要重复时，可以简洁表达为“～です”。
   北京支社は　何時に　始まりますか。
   ————8時です。
3. 毎朝　何時に　起きますか
   当两个名词都表示时间，而前面的名词中包含“毎”时，两个名词之间不加“の”。当前面的名词中不包括“毎”时，可以加“の”，也可以不加“の”。
4. ～ごろ
   “ごろ”后面一般不加“に”。
第6课：
1. たしか
3. “家” 和 “うち”
   指建筑物时一般用“家”，着重指家人时一般用“うち”。
4. まっすぐ　帰りました
5. “それ”的用法
   毎日　夜　11時まで　働きます。
```
