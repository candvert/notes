# 会话
## 第1课
```
搭话·打招呼
搭话有「すみません，～」「あのう，すみません。～」「ちょっと，すみません。～」「あのう，失礼ですが，～」
回答有「はい？」「何でしょうか？」「あっ，どうも。」
  与曾经见过面、有点印象的人打招呼，确认其姓名时，先说“失礼ですが”或者“あのう，失礼ですが”。
  失礼ですが，田中さんですか。
2. 先ほど
   “先ほど”是“さっき”的礼貌说法。
3. 先ほどの方　「助词“の”」
4. 助かりました
   “助かります”是他动词“助けます”的自动词形式。可以用助词“が”来表示获得帮助的人。
   仕事を手伝えば，お父さんが助かります。
   “助かります”有时还表示感谢的意思。
   荷物を持っていただいて，助かりました。
5. 感じがします
   “感じます”多表示直接的感受，而“感じがします”多表示留有某种印象。
   勉強すればするほど，難しくなる感じがします。
   この景色を見ると，日本に来たという感じがします。
6. “～へは”与“～は”
   助词“が”　“を”后续助词“は”时，需要去掉“が”　“を”变成“は”。助词“へ”后续“は”时，“へ”可以省略也可以不省略，而助词“に”　“で”　“と”　“から”　“より”等后续“は”时，成为“には”　“では”　“とは”　“からは”　“よりは”等形式。
   外からは中にだれがいるのか分かりません。
7. 転勤になった
8. “すっかり”与“ずいぶん”
   空港がすっかり変わりましたね。驚きましたよ。
   “ずいぶん”，如“ずいぶん歩きました”一样，也可以表示动作的量，或像“ずいぶん新しい”那样修饰形容词。
   ずいぶんたくさん買い物をしましたね。
9. ついていけない
   “ついていけない”是动词“ついていきます”的可能形式“ついていけます”对应的否定形式“ついていけません”的简体形式。
   あの人の考え方にはついていけません。
10. ～くらいです    （强调程度）
   ～くらい/～ぐらい + の + 名词
   財布を忘れましたが，ジュースを買うくらいのお金は持っています。
11. “～てきました”与“～た”
   “～てきました”表示变化的开始，或不断朝着某个结果进展的状态。
   日本語の発音に少し慣れてきました。
   救急車のサイレンの音がだんだん大きくなってきました。
   下面的例句表示确认建筑物一点一点地逐渐进入视线的过程。
   ほら，高層ビルが見えてきたでしょう？
   与上例相比，下面这个例句的重点不在于变化，而只是单纯地确认是否看得见。
   高層ビルが見えたでしょう？
```

# 课文
## 第一课
```
1. ～とは
   “～とは”是“～というのは”的省略形式。
   JRとは，Japan Railwaysの略です。
   “～とは”还有下定义的用法，表现形式为“～とは～のことです”。
   おせち料理とは，正月に食べるお祝いの料理のことです。
   与“～とは”说法类似的有“～というものは”　“～ということは”。“～というものは”前接事物名词，用于说明该事物的本质特征。“～ということは”前接行为动词，用于说明该行为的本质或性质特征。
   人間というのものは1人で生きていくことはできません。
   子供を育てるということは，とても楽しいことです。
   此外，还有前接小句的用法，根据发生的事件推测该事件发生的原因或归纳出符合逻辑的结果。
   彼が怒ったということは，きっとひどいことを言われたのでしょう。
2. ～的
   表示抽象意义的汉语词后续“的”，作为二类形容词使用。主要有三种意思：表示场所范围，如“世界的”　“国際的”等；表示具有某种性质或倾向，如“消極的”　“積極的”等；表示某种状态，如“具体的”　“抽象的”等。
   世界的に有名な新幹線もJRの路線の1つで，日本の主要な都市と都市を結んでいます。
3. ～は～の1つです
   “AはBの1つです”表示A属于B范围当中的一种。
   日本はアジアの国の1つです。
4. 都市と都市を結ぶ
   用于句尾时，使用“結んでいます”的形式表示状态。
   中国旅行は，わたしと妹を結ぶ大切な思い出です。
   化石は，古い時代と現代を結んでいます。
5. どんどん    （表示样子的副词）
   佐藤さんの中国語はどんどん上手になっています。
6. 大～
   据后续词不同，“大”的读法有“だい”　“たい”　“おお”三种。读作“だい”时，后续词一般为两个字的汉语词，如“大事件”　“大地震”　“大成功”等。读作“たい”时，“大”一般已经成了词语的一部分，不能分离，如“大変”　“大切”等。读作“おお”时，后续词一般为日语固有的词，如“大金持ち”　“大雨”　“大汗”等。
   另外，只有“大規模”是二类形容词，常用“大規模な”的形式修饰名词，但一般不用“大規模だ”的形式结句。
   中国では大規模な都市開発計画が進んでいます。
7. また
8. “もの”的用法
   “もの”和汉语的“的”一样，为避免词汇重复，用来代替前面出现的名词。
   公営の鉄道は，地方自治体などが経営するものです。
9. ～化
   “名词+化”表示改变成该名词表示的那种状态。由于表示变化的意思，所以既可以用作名词也可以后续“する”用作三类动词。
   世界で初めて実用化されたリニアモーターカーも国営です。
10. “上り”和“下り”
   “上り”是“上ります”去掉“ます”的形式，用作名词，表示“从低处往高处移动”；“下り”是“下ります”去掉“ます”的形式，作名词使用，表示“从高处往低处移动”。
   另外，和“上り”　“下り”一样，部分动词的“ます形”去掉“ます”后的形式可以作为名词使用，如“行きはタクシーで行きます”。另外还可以与名词结合在一起使用，如“東京行き”　“雨降り”　“通行止め”。但是这种用法只适用于少数动词，而不是所有动词都适用。
   北京行きの航空便は1日に8便あります。
11. そこで
12. こんな    （指示上下文内容）
   “こんな”是指示词“このような”的比较随便的说法。
   そこで，こんななぞなぞがあります。「日本でいちばん高い駅はどこでしょう。」
```
