# 会话
## 第1课
```
搭话·打招呼
   搭话有「すみません，～」「あのう，すみません。～」「ちょっと，すみません。～」「あのう，失礼ですが，～」
   回答有「はい？」「何でしょうか？」「あっ，どうも。」
   与曾经见过面、有点印象的人打招呼，确认其姓名时，先说“失礼ですが”或者“あのう，失礼ですが”。
   失礼ですが，田中さんですか。
2. 先ほど
   “先ほど”是“さっき”的礼貌说法。
3. 先ほどの方　「助词“の”」
4. 助かりました
   “助かります”是他动词“助けます”的自动词形式。可以用助词“が”来表示获得帮助的人。
   仕事を手伝えば，お父さんが助かります。
   “助かります”有时还表示感谢的意思。
   荷物を持っていただいて，助かりました。
5. 感じがします
   “感じます”多表示直接的感受，而“感じがします”多表示留有某种印象。
   勉強すればするほど，難しくなる感じがします。
   この景色を見ると，日本に来たという感じがします。
6. “～へは”与“～は”
   助词“が”　“を”后续助词“は”时，需要去掉“が”　“を”变成“は”。助词“へ”后续“は”时，“へ”可以省略也可以不省略，而助词“に”　“で”　“と”　“から”　“より”等后续“は”时，成为“には”　“では”　“とは”　“からは”　“よりは”等形式。
   外からは中にだれがいるのか分かりません。
7. 転勤になった
8. “すっかり”与“ずいぶん”
   空港がすっかり変わりましたね。驚きましたよ。
   “ずいぶん”，如“ずいぶん歩きました”一样，也可以表示动作的量，或像“ずいぶん新しい”那样修饰形容词。
   ずいぶんたくさん買い物をしましたね。
9. ついていけない
   “ついていけない”是动词“ついていきます”的可能形式“ついていけます”对应的否定形式“ついていけません”的简体形式。
   あの人の考え方にはついていけません。
10. ～くらいです    （强调程度）
   ～くらい/～ぐらい + の + 名词
   財布を忘れましたが，ジュースを買うくらいのお金は持っています。
11. “～てきました”与“～た”
   “～てきました”表示变化的开始，或不断朝着某个结果进展的状态。
   日本語の発音に少し慣れてきました。
   救急車のサイレンの音がだんだん大きくなってきました。
   下面的例句表示确认建筑物一点一点地逐渐进入视线的过程。
   ほら，高層ビルが見えてきたでしょう？
   与上例相比，下面这个例句的重点不在于变化，而只是单纯地确认是否看得见。
   高層ビルが見えたでしょう？
```
## 第2课
```
开始交谈·结束交谈
   （1）对象为久未见面的人
   “（お）久しぶり”　“（お）久しぶりですね”　“ごぶさたしています”
   どうも，お久しぶりです。お元気でしたか。
   （2）对象为最近刚见过面的人
   “この間はどうも”　“先日はありがとうございました”　“その節はお世話になりました”
   先日は結構な物をありがとうございました。
   先日はごちそうさまでした。
   先日はお邪魔しました。
   （3）对象为初次见面的人
   可以先说“はじめまして”作为问候语，然后做简单的自我介绍。
   （4）想与对方交谈，对方正忙着其他事情
   “お話中，失礼します”　“お仕事中，ちょっと失礼します”　“お電話中，すみません”
   想与对方交谈，而对方正与他人说话或正忙于其他的事情，这时要用“～中，失礼します”或者“～中，申し訳ありません”　“～中，すみません”等表达方式。有时还要加上“あのう”或“ちょっと”等词语。
   （5）结束交谈
   “では，また”　“じゃあ，また”　“では，失礼します”　“では，これで”　“今日はこれで”
   “じゃあ，ここで”是“じゃあ，ここで失礼します”的省略说法。有时也说“今日はこれで（失礼します）”。
1. ～ぶり
   修饰动词使用“～ぶりに+动词”的形式；修饰名词使用“～ぶり+の+名词”的形式。“～ぶり”不能用于比“～日”更短的。“～日”也只能用于两天以上，而不说“1日ぶり”。
   佐藤さんといっしょに仕事をするのは，4年ぶりですね。
2. ～ものです    （感慨）
   使用“小句（简体形式）+ものです”的形式。二类形容词使用“二类形容词+なものです”。
   早いものですね。
   平和な世界になってほしいものです。
3. その節
   没有“あの節”　“この節”的用法。
   その節はお世話になりました。
   その節は結構な物をいただきまして，ありがとうございました。
4. いらっしゃいました
   “来ます”的尊敬表达方式有“いらっしゃいます”　“来られます”等。
   JC企画の李さんがいらっしゃいました。
5. こちらにお願いします
   コーヒーをお願いします。
   サインをお願いします。
6. 今度
7. 介绍某人
   こんにちは，李さん，紹介します。こちらは，今度の「金星」プロジェクトの責任者で，佐藤です。
   佐藤さん，こちらは，JC企画の李さんです。
8. 公司内部人员的称谓
```
## 第3课
```
自我介绍
   “王風と申します”是“王風といいます”的礼貌说法。
   介绍对自己的称呼时，说“⚪⚪と呼んでください”。
   自我介绍时常用自谦的表达方式，例如不说“王風といいます”，而说成“王風と申します”，以及“～を担当させていだきます”　“～を担当しております”等等。
1. ～を含めて
   今回の「金星」プロジェクトは，私を含めて4名のスタッフが担当いたします。
2. ～名
   “名”比“人”更为庄重，大都用在正式的场合。
   このヨットの定員は7名です。
3. 主に
   “主”是二类形容词，后续动词时用“主に”，后续名词时用“主な”。此外，还有“主として”的用法，但不能后续名词，这时读“しゅとして”。
   主に営業を担当させていただきます。
   田中先生は主として初級のクラスを担当しています。
4. よろしくお願いいたします
   “いたします”是“します”的自谦语。更礼貌的说法还有“よろしくお願い申し上げます”。
5. ～にとって，～
   張さんにとって，仕事は人生のすべてではありません。
   与表示观点的“～にとって”不同，“～として”是站在同一角度，表示“处于～的立场”　“属于～范畴”的用法。“として”后续名词时需要加“の”。
   困っている人を助けるのは，人間として当然のことです。
   “～としては”前接表示人物或组织的名词，表示“从该角度考虑”。
   彼女としてはとても辛かったようです。
   “～にしたら/～にしてみれば/～にすれば”前接表示人的名词，表示“对于该人来说”的意思。
   彼にしてみれば，こんな朝早く出かけるのは嫌でしょう。
6. 使用“～が，～”　“～けど，～”的说法    （铺垫）
   “～けど”只用于口语，不用于书面语。
   すみませんが，今お話してもいいでしょうか。
7. 今でも
   “今も”比“今でも”要稍显正式一些。
   紹興で飲んだ紹興酒の味は今でも覚えていますよ。
```
## 第4课

```
转述信息
   “小句（简体形式）+そうです/ということです/とのことです”“名词/二类形容词+だそうです/だということです/だとのことです”都是转述听到的信息时使用的表达方式。
   “～そうです”需要明确信息来源时，可以在信息来源后使用“～によれば”　“～によると”　“～の話では”等形式，但不用“某人+が”的形式。
   “～とのことです”　“～ということです”句尾使用过去形式“た”还可以表示以前听说过的事情。
   山田さんは中国の事情に詳しいからということでした。
   “～と言っていました”敬语表达方式是“～とおっしゃっていました”。
   交谈的对方要求传话时，可以回答说“そうお伝えします”　“そのようにお伝えします”等。
   与朋友、家人等关系密切的人说话，还可以用“小句（简体形式）+って”的形式来表示传闻。
1. うち
   うちの青木主任がちょっとお話をしたいと言っていました。
2. わたしにですか
   わたしに（話したいの）ですか。
3. ご都合
   提出疑问时用“ご都合はいかがですか”　“ご都合はどうですか”，回答时用“都合がいいです”“都合が悪いです”。
4. ～ところ
   还可以后续“を”，用“ところを”的形式。
   お忙しいところ，すみません。
   お休みのところ，申し訳ありません。
   お疲れのところを，手伝ってもらって，すみません。
5. 実は
6. 思ったより
   除“思う”外，“より”还可以前接“予想”　“想像”等词。前接动词时，动词需要变为“た形”或“～ていた”的形式；前接名词时则直接使用“名词+より”的形式。类似的表达方式有“～以上に”。
   思ったより大変な仕事になりそうなんです。
   アフリカ旅行は思った以上に楽しかったです。
7. 突然ですが    （铺垫）
8. ～てもらえませんか
   “～てもらいます”　“～てくれます”是表示受益的表达方式。
9. 省略
   わたしが「金星」プロジェクトの担当者に？
10. 事情
   山田さんは中国の事情に詳しいからということでした。
11. ～し，～から，～
   もうこんな時間ですし，明日も早いので，そろそろ失礼します。
   陳さんは日本語もうまいし，英語もできるから，安心です。
12. 分かりました    （分かる）
   更加明确地表示同意时，可以使用“承知しました”，“承知しました”是自谦语。另外，电子邮件等书面语中有时使用“了解しました”。
```
## 第5课
```
转移话题
   ところで
   話が変わりますが/話が飛びますが
   さて
   けれども/でも
   あっ，それから/あと
   それで/で
   それでは/では，（次の話題に行きましょう）/次に
1. 最～
   “最大”　“最小”　“最適”等，修饰名词时后续“の”，也有像“最適な”这样的二类形容词。
   太郎はわたしの最愛の息子です。
2. ～をきっかけにして
   使用“～を+特定名词+に+して”的形式，表示方式，如“～をきっかけにして”　“～を手がかりにして”　“～を頼りにして”。有时省略“して”。
   山本さんは就職をきっかけに（して）一人暮らしを始めた。
   李さんが書いてくれた地図を頼りに（して），美術館を探した。
3. 助词“も”
   “50人も”这种数字后续“も”的用法表示数目多。
   最近では，海外からの注文も増えてきています。
4. ～に受ける
   “表示人的名词+に受ける”的形式表示“该人”对此很中意、评价好、喜欢等意思。
   「金星」という名前は，中国人に受けるでしょうか。
5. “が”与“の”的变换
   在修饰名词的小句内部，有时可以将“が”改成“の”，如：“車が通る道”可以变成“車の通る道”。表示一般的性质时多用“の”。
   日本と中国とでは，意味の違う漢字があると聞きましたが。
   陈述发生的特定事件时多用“が”。
   さっき車が2台通った道。
6. “～と”与“～ば”    （假定条件）
   「きんぼし」と読むと，意味が違いますよね。
   “～ば”后项为表示命令或意志的句子时，“～たら”比“～ば”用得多。
   もし機械が故障したら知らせてください。
7. “力士”　“横綱”与“金星をあげる”
   以相扑为职业的人在日语中叫做“力士”。此外，取“相撲を取る人”的意思，又叫做“相撲取り”。其他的运动项目一般称作“～選手”，如“野球選手”　“柔道選手”等，但没有“相撲選手”的说法。
   “金星をあげる”的说法用于称赞某人的特殊功绩。
8. ～ということからきた～
   用于句尾时使用“～からきています”的形式。相近的表达方式还有“～に由来した～”。
   駅弁は，「駅で売っている弁当」ということからきた言葉です。
9. ～って
   “～って”是“～と”　“～とは”　“～という人は”等在会话中的缩略形式。
   「金星」って，ボトルの形がすてきですね。
12. 人気があるんです。
13. ～のは間違いない
   “の”前接动词简体形式使其名词化。“の”可以用“こと”替代，另外还有一种说法是“～ことに間違いない”。
   このボトルなら，中国でも売れるのは間違いないですね。
   “小句（简体形式）+に違いない”和“小句（简体形式）+に相違ない”都表示推测，可以与“きっと”　“たぶん”等副词一起使用。“～に相違ない”只用于书面语。二类形容词和名词用“二类形容词/名词+に違いない/に相違ない”的形式。
   このガイドブックは内容が豊富でおもしろい。きっと売れるに違いない。
   “相違ない”还可以单独使用，表示所说的是正确的，用于书面语。
   以上のとおり，相違ありません。
14. それを聞いて    （指示上下文内容）
   それを聞いて安心しました。
```

# 课文
## 第一课
```
1. ～とは
   “～とは”是“～というのは”的省略形式。
   JRとは，Japan Railwaysの略です。
   “～とは”还有下定义的用法，表现形式为“～とは～のことです”。
   おせち料理とは，正月に食べるお祝いの料理のことです。
   与“～とは”说法类似的有“～というものは”　“～ということは”。“～というものは”前接事物名词，用于说明该事物的本质特征。“～ということは”前接行为动词，用于说明该行为的本质或性质特征。
   人間というのものは1人で生きていくことはできません。
   子供を育てるということは，とても楽しいことです。
   此外，还有前接小句的用法，根据发生的事件推测该事件发生的原因或归纳出符合逻辑的结果。
   彼が怒ったということは，きっとひどいことを言われたのでしょう。
2. ～的
   表示抽象意义的汉语词后续“的”，作为二类形容词使用。主要有三种意思：表示场所范围，如“世界的”　“国際的”等；表示具有某种性质或倾向，如“消極的”　“積極的”等；表示某种状态，如“具体的”　“抽象的”等。
   世界的に有名な新幹線もJRの路線の1つで，日本の主要な都市と都市を結んでいます。
3. ～は～の1つです
   “AはBの1つです”表示A属于B范围当中的一种。
   日本はアジアの国の1つです。
4. 都市と都市を結ぶ
   用于句尾时，使用“結んでいます”的形式表示状态。
   中国旅行は，わたしと妹を結ぶ大切な思い出です。
   化石は，古い時代と現代を結んでいます。
5. どんどん    （表示样子的副词）
   佐藤さんの中国語はどんどん上手になっています。
6. 大～
   据后续词不同，“大”的读法有“だい”　“たい”　“おお”三种。读作“だい”时，后续词一般为两个字的汉语词，如“大事件”　“大地震”　“大成功”等。读作“たい”时，“大”一般已经成了词语的一部分，不能分离，如“大変”　“大切”等。读作“おお”时，后续词一般为日语固有的词，如“大金持ち”　“大雨”　“大汗”等。
   另外，只有“大規模”是二类形容词，常用“大規模な”的形式修饰名词，但一般不用“大規模だ”的形式结句。
   中国では大規模な都市開発計画が進んでいます。
7. また
8. “もの”的用法
   “もの”和汉语的“的”一样，为避免词汇重复，用来代替前面出现的名词。
   公営の鉄道は，地方自治体などが経営するものです。
9. ～化
   “名词+化”表示改变成该名词表示的那种状态。由于表示变化的意思，所以既可以用作名词也可以后续“する”用作三类动词。
   世界で初めて実用化されたリニアモーターカーも国営です。
10. “上り”和“下り”
   “上り”是“上ります”去掉“ます”的形式，用作名词，表示“从低处往高处移动”；“下り”是“下ります”去掉“ます”的形式，作名词使用，表示“从高处往低处移动”。
   另外，和“上り”　“下り”一样，部分动词的“ます形”去掉“ます”后的形式可以作为名词使用，如“行きはタクシーで行きます”。另外还可以与名词结合在一起使用，如“東京行き”　“雨降り”　“通行止め”。但是这种用法只适用于少数动词，而不是所有动词都适用。
   北京行きの航空便は1日に8便あります。
11. そこで
12. こんな    （指示上下文内容）
   “こんな”是指示词“このような”的比较随便的说法。
   そこで，こんななぞなぞがあります。「日本でいちばん高い駅はどこでしょう。」
```
## 第二课
```
1. ～よりも
   “よりも”意义和“より”相同，但比较的语气更强一些。
   日本人は，握手よりもお辞儀をするのが一般的です。
   “より”还有表示起点的用法，如“会議は3時より始めます”，这种用法不能后续“も”。
2. ～から～まで
   表示事物的范围。
   この工場では，ゴムの製造から加工まで行っています。
3. お辞儀をし，～
   口语中按事件发生的先后顺序叙述时，经常使用“～て，～”的形式。书面语中一般不用“～て，～”，而用动词“ます形”去掉“ます”的形式连接。
   深いお辞儀をし，相手の名刺をお辞儀をしながら受け取ります。
   但是，当前一个动词和后一个动词的动作几乎同时发生或者前一个动词修饰后一个动词时，通常使用“～て～”的形式。如“持っていきます”　“歩いて来ます”　“帽子をかぶって出かけます”一般不能改为“持ちいきます”　“歩き来ます”　“かぶり出かけます”的形式。
4. ～ほうが～です
   使用“小句（简体形式）+ほうが+形容词です”的形式。在使用动词“ない形”时，不能使用“～なかったほうが～”的形式。
   名刺を出す時も受け取る時も両手を使うほうが丁寧です。
5. 仕事の場
   特に仕事の場で会った相手には，個人的な質問は避けたほうがいいでしょう。
6. ～ほうがいいでしょう
   “动词た形+ほうがいい”是向对方提出某种建议时使用的表达方式。
   栄養があるものをたくさん食べたほうがいいでしょう。
7. ～か，～か，～
   不含疑问词的疑问句一般使用“～かどうか”的形式。
   結婚しているかどうか，給料はいくらか，年はいくつかなどは普通は聞きません。
   昨日どこへ行ったか，だれと会ったか，何をしたかなど，いろいろ聞かれました。
8. 年
   わたしは兄と年が離れています。
   和数字一起表示年龄时，用“歳”而不用“年”，如“今年40歳になりました”。手写时有时用汉字“才”代替“歳”。
9. いくら～ても～
   表达同样意思的表达方式还有“いかにAてもB”　“どんなにAてもB”　“どれだけAてもB”。“いかにAてもB”比较正式，多用于书面语。
   いくら早く答えても間違いがあれば合格できません。
   どんなに遅くなっても大丈夫です。
10. 怖い顔をして
   “表情形容词+顔をする”是表示面部表情的表达方式。“顔をする”还可以和表示情感的动词一起使用，如“びっくりしたような顔をする”　“あきれた顔をして”等。
   彼女はわたしを見ると，びっくりしたような顔をしました。
11. ～ては/～では
   会话中有时把“ては”说成“ちゃ（あ）”，把“では”说成“じゃ（あ）”。
   あんな人が教師では学生がかわいそうです。
   こんなに給料が安くては困ります。
12. 気持ちがこもっています
   “こもります”的意思是充满某物而不外露。除“気持ち”外，还可以用“思い”“真心”等。
   気持ちがこもっているかどうかは，予想以上に相手に伝わってしまうものです。
   “こもります”是自动词，还有用他动词“こめます”的表达方式，如“気持ちをこめます”　“思いをこめます”等。
   彼らは結婚のお祝いの気持ちをこめて歌を歌いました。
13. ～以上に
   “动词（基本形/た形）+以上に”　“名词+以上に”。前接动词基本形时，主句不能使用过去形式。
   わたしたちが予想する以上に人が集まったら，どうしましょうか。
14. ～ものです    （性质）
   前接动词、一类形容词的基本形以及二类形容词后续“な”的形式。
   表示本质时，否定形式为“～ない+ものです”；表示理所当然时，否定形式为“～ものではありません”　“～ものではないです”。在口语中，有时也用“～もんじゃないです”的形式。
   人の心は分からないものです。
   そんな失礼なことは言うものではありません。
```

```
1. 多くの
   将一类形容词词尾“い”变为“く”再加“の”的用法还有“遠い->遠くの”　“近い->近くの”等。另外，“少ない”要变成“少しの”。其他的一类形容词一律直接后续名词。
```

## 第四课
```
1. かつて
   口语中有时音变成“かって”。
2. ～といえば
3. 当たり前
   不能用于句首。
4. 取り組み
   “取り組み”是动词“取り組みます”去掉“ます”的形式，用作名词。
   日本政府の環境問題への取り組みはまだ始まったばかりです。
5. ～てきます
   “动词て形+きます/きました”表示某种变化或动作不断发展或处于某种状态。“动词て形+きています”的形式表示从过去一直持续到现在。
   世界の人口は増え続けてきました。
   この店は昔から食器を扱ってきています。
   表达变化或动作一直在持续的表达方式还有“～ています”　“～ていきます”。“～ていきます”有时还用作“～てゆきます”。
   世界の人口はこれからも増加してゆきます。
6. 1980年代
   “～年代”与汉语相同。另外，“10代”指10岁到19岁，“20代”指20岁到29岁。
   10代の子供たちに，大人の生活習慣病が広がっています。
7. さらに
8. フレックスタイム制度
9. バランスを取ります
   类似的表达方式还有“バランスを保ちます”。消极意义的表达方式有“バランスが崩れます”　“バランスを失います”等。
   収支のバランスが崩れて，会社は倒産した。
10. これまで
   “これまで”只用于较大时间范围的过去。
11. ほとんど
   “ほとんど”还可以作名词使用。既可以用“ほとんどが～”的形式充当主语，也可以后续助词“の”修饰名词，如“ほとんどの人”。此外，意思相近的“だいたい”也可以作副词和名词使用，作名词时可以后续“が”　“の”　“を”　“に”等助词。
   ほとんどの学生が，朝ご飯を食べないで学校に来ます。
12. 出世
   “出世”是名词，意思相当于汉语的“出息”　“发迹”，可以后续“する”作动词使用。通常指事业上的身份提升。“出世の妨げ”意思是阻碍或干扰晋升的因素。
13. ～（の）ではないか
   接续形式为“简体形式+のではないか”“二类形容词+のではないか”“名词+ではないか”。“～（の）ではないか”用于书面语。
   出世の妨げとなるのではないかと考える人が多かったからです。
14. 見られます
   “見られます”是“見ます”的可能形式。
   携帯電話でインターネットを楽しむ人がよく見られます。
15. “これ”　“この”与“このように”    （指示上下文内容）
   さらに，フレックスタイム制度を導入する会社も増えています。これは，自分の出勤と退勤の時間を自由に決められる制度です。
   日本の山では富士山が有名です。この山には毎年30万人が登ります。
   このように，日本のサラリーマンの働き方は，かつてに比べると大きく変化しています。
   指示后述内容：
   これはわたしの個人的な意見ですが，佐藤さんの考え方は間違っていると思います。
   こんな話があるのを知っていますか。100年後，日本の人口は今の約半分になるという予測が発表されたのです。
16. 時間に追われます
   习惯上使用被动形式。
17. 生活を送ります
   “生活を送ります”是书面语，有使人感觉到时间流逝的语感。口语中多用“暮らします”。
   都会での時間に追われる生活を送ることをやめて，田舎に引っ越す人もいます。
18. 都会から田舎に引っ越す
```

## 第五课
```
1. ～という点から
   日本語の単語を，元の言葉は何かという点から分類すると，・・・
   遅くまで開いているという点で，あの店は便利だ。
2. もともと
   类似的词语还有“元来”　“本来”。
4. ～に基づいた～
   有“～に基づいた+名词”　“～に基づいて+谓语”　“～に基づく+名词”等表达方式。用于句尾时一般使用“～に基づいています”的形式。“～に基づいた～”和“～を基にした～”意思相同。而且，前接名词时“基づく”和“基づいた”的意思相同。
   これまでの経験に基づいて判断するのは間違っている。
   今回の調査は法律に基づいている。
5. ～を中心とした～
   与“～を中心とする～”意思相同。
   この雑誌は20代の女性を中心とする若者に読まれている。
   后续谓语动词时使用“～を中心に+动词”　“～を中心として+动词”的形式。
   この書店は中国の本を中心に扱っている。
6. ～のうち
   航空便で送ったグラスのうち，半分が割れていた。
7. ～を～と呼ぶ
   土曜日と日曜日など，1週間に2日休日があることを週休2日制と呼ぶ。
8. たいてい
   たいていの人は，畳の上に布団を敷いて寝るような部屋を想像する。
   平日の夜や休みの日には，たいてい家で読書をしている。
9. ～ような～/～ように～
   “名词+の+ような～”　“二类形容词+な+ような～”　“小句（简体形式）+ような～”。后续谓语成分时使用“～のように”的形式。
   この子供がかいたような絵は1億円もするらしい。
   東京のようにたくさんの人が住む都会に行きたい。
   鳥が飛ぶように空を飛んでみたい。
   “～というような/～というように”和“～といったような/～といったように”
   “～といったような”有时还以“～といった”的形式出现。此外，跟在谓语后面时，变为“～というように”的形式。
   エビ，イカ，卵といったようなコレステロールの高い食品は避けたほうがいい。
   テレビ，冷蔵庫，布団というように，一人暮らしで必要な物を買った。
   列举一个例子时使用“～のような”，不能使用“～というような”　“～といったような”　“～といった”的形式。
   エビのようなコレステロールの高い食品。
   ～というふうな/～というふうに
   北京，上海，広州というふうに，中国の大きな都市を順番に訪問した。
```
