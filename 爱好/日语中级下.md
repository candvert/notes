# 会话
## 第17课
```
称赞·谦虚
   名演奏でした。感動しました。とても言葉では表せません。
   いえ，そんな・・・たいしたことではありません。
   ありがとうございます。でも，まだまだ不十分なところもあります。
   すばらしい結果ですね。
   ————ありがとうございます。そう言っていただけると本当にありがたいです。
1. 省略主语
   以上が，ホームページの「「金星」の故郷を探る」で取り上げる記事の一例です。
2. ～抜きで
   これはお世辞抜きでおもしろいですよ。
   还常常使用“～は抜きにして”的表达方式。
   堅苦しいあいさつは抜きにして思いきり飲みましょう。
3. ほかにも
   ほかにももう1つ提案があるんですが・・・。
4. 確かに～でも～
   ええ，確かに。でも，それは日本国内でのことですよ。
5. 売り込む    （复合动词）
   湖にごみを投げ込むのはやめてください。
6. ～の間
   若者たちの間でこのカクテルの人気が出たら，「金星」の売り上げにもつながるんじゃないでしょうか。
7. たいした
   これはわたしがかいた絵で，たいした作品ではありません。
8. 私ども
   中国用のネーミングやラベルのデザインは，私どもで引き受けます。
```
## 第18课
```
协调意见分歧
   正直に言えば，この報告書のデータはすべて正しいわけではありません。
   何というか，うまく言えませんが，ほかにもっといいアイデアがありそうに思います。
1. 对顾客的用语
   いかがでしょうか？お気に召しましたでしょうか？
2. とおっしゃいますと
   没听清对方的话时可用：
   すみません，何とおっしゃいましたか？
   すみません，もう一度ゆっくりおっしゃってくださいませんか？
5. 多くなってきている
   最近，作業のミスが増えてきてしまっている。
6. 好まれる
   日本酒は日本人に好まれると思う。
7. ～とおりだ
   確かにおっしゃるとおりです。
   わたしが説明したとおりにやれば，この故障は直せます。
9. ～を～にした～/～を～とした～
   弊社では，「金星」をベースにした新感覚のカクテルもご用意しています。
   この服は「エコロジー」をコンセプトとした素材でできています。
11. こちらの青いほう
   子供は2人で，上のほうが娘で，下のほうが息子です。
12. 置く
   あの店は中国製のいい筆を置いているので，時々買いに行きます。
```
## 第19课
```
道歉
   ご迷惑をおかけして，本当に申し訳ありませんでした。
   お怒りはごもっともです。二度とこんなことがないように，気をつけます。
   すぐにお詫びにお伺いします。
1. “ね”的用法
   実はね，お宅の「金星」なんですが，困ってしまいましたよ。
3. “どういったこと”　与　“どういうこと”
   どういったことでしょうか。
5. ～にひびが入る
   彼は，彼女との関係にひびが入って悩んでいるらしい。
6. ～ようにしてください
   週末に間に合うようにしてください。
7. ～次第
   分かり次第ご連絡を差し上げます。
8. いいですよ
   原因なんて，いいですよ。
9. ～限り
   契約は今月限りにしてほしいと言っています。
12. ～のほう
   ご注文のほうはお決まりですか。
13. かんかん
   とにかく，担当者がかんかんなんです。
14. わざわざ
   わざわざ来ていただかなくても結構です。
```
## 第20课
```
鼓励
   わたしにできることがあれば何でも言ってください。
3. ～なら
   佐藤さんなら，外灘にいるんじゃないかって教えてくれました。
5. 考え事をする
   車を運転しながら考え事をしていて，人をひきそうになった。
6. ここのところ
   ここのところ，失敗だらけなんです。
7. ～だらけ
   間違いだらけの答案を採点する。
8. やっと
   家族3人がやっと暮らしていけるぐらいの収入しかありません。
10. ～と思ったら，～
   その学生は，就職したと思ったらさっさと辞めてしまった。
13. ～を前にする
   わたしは大勢の人を前にすると，ひどく緊張するんです。
14. しょんぼりする
   太郎は母親の帰りを1人でしょんぼりと持っていた。
15. 変に思う
   恋人同士なんだから手をつないで歩いても，だれも変に思いませんよ。
16. せっかく～のに
   せっかくいいニュースを持ってきたのに。
   せっかく注文を追加したのだから，全部食べてください。
19. ～ですって
   “小句 + だって”的礼貌表达方式为“～ですって”。
   何ですって？
```

## 第十七课
```
3. 無数に
   公園の周囲には「胡同」と呼ばれる狭い路地が無数に存在している。
4. さかのぼる
   「胡同」を知るには，歴史を約700年前までさかのぼらなければならない。
5. 通る
   この住居を取り囲むように通っているのが「胡同」である。
6. 東西南北
7. せいぜい
   経験があるといってもせいぜい2，3年だ。
8. ～とまでいわれた
   蘇州は美しい街だ。東洋のベニスとまでいわれている。
9. ～ことに
   残念なことに，近代化が進み，胡同の数が急激に減ってきてしまった。
10. ～（よう）とする
   小鳥はかごから逃げようとしている。
   この泉の水は枯れようとしている。

```

```
   北上貿易の皆様におかれましてはお元気でお過ごしのことと存じます。
4. 助词“で”    （动作主体）
   警察では目撃者がいなかったか調査をしている。
7. “急なお願い”　与　“勝手なお願い”
   急なお願いで申し訳ございませんが，近いうちに一度お目にかかり，お話を伺えればと存じます。
   勝手なお願いでご迷惑をおかけいたしますが，よろしくご検討くださいますようお願い申し上げます。

   みんなが寝ているうちに（＝間に），こっそり出て行った。
   夜のうちは，ここから動かないほうがいいでしょう。
9. 一度お目にかかる
   “～にお目にかかる”是“～に会う”的自谦语。
   一度，仕事で，木村先生にお目にかかったことがある。

   お近くにいらっしゃる折には，我が家にもお寄りください。

15. ～申し上げる
   これから，お客様を自動車で駅までお送り申し上げます。
16. この度
   この度の地震では，100人以上が亡くなりました。

   ご不明の点がございましたら，お気軽にお問い合わせください。
```

## 第十九课
```
1. 代名詞
   東京の水道水は，かつて「まずい水」の代名詞だった。
2. 苦情が寄せられる
   多くの視聴者から番組に対する意見や苦情が寄せられた。
3. ～つつある
   ところが，この水道水が変わりつつある。
4. 大手
   加藤さんの息子さんは，大手の商社に就職が決まったらしい。
5. ～には～とある
   ラベルには「原材料名：水（水道水）」とある。
   この学校の規定には，無断で欠席すると退学とある。
6. ～と評判だ
   都内の浄水場で処理した水をボトルに詰めて販売していて，味もおいしいと評判だ。
7. ～ならびに～
   本日の議会では，新しい法案ならびに予算案が可決されました。
8. ～に伴って～
   都市の緑化に伴って，駅周辺に花や木が増えてきた。
9. ～のもと（で）
   国連の旗のもと，各国が協力して紛争の平和的な解決を目指している。
   この機関は3つの原則のもとで活動している。
10. ～のもとになる～
   鉄のもとになる鉄鉱石はすべて輸入に頼っています。
11. ～一方（で），～    （同时进行）
   彼は事故の原因を調べる一方（＝調べながら），被害の拡大を防いだ。
   その知らせを聞いて，驚いたと同時にとてもうれしかった。
```
