## 日语输入法
```
按两个n打出ん
促音っ按下一个假名的首字母的两次或xtsu打出     比如北海道/ほっかいどう需要按下hokkaidou
假名注音前加x打出小写   比如ディ需要按下dexi
ヴ   vu
英日切换Shift + CapsLock
F7切换为片假名
F6切换为平假名
```
## 发音
```
“た”行的“た ta” “て te” “と to”的辅音与汉语“t” “d”的发音相当。出现在词头时，与“t”相近，出现在词中时，与“d”相近。基于同样的道理，“ち”出现在词头和词中时，分别相当于汉语的“qi” “ji”，“つ”出现在词头和词中时，分别相当于汉语的“ci” “zi”。
```

## 动词
```
   日语的动词可分为自动词和他动词。最典型的他动词具有主语作用于宾语而使其发生变化的含义，其宾语用“を”表示。自动词是不带宾语的动词，表示不考虑外力影响的情况下，主语自主地进行动作，或自然地发生变化，其主语用“が”表示。

一类动词（五段动词）
1. 结尾是う段非る的动词，肯定是五段动词
2. あ段+る、う段+る、お段+る
3. 特例 切る/きる　帰る/かえる　走る/はしる

ます形：う段 - い段 + ます

て形：
す - して              話す - 話して
く - いて              磨く - 磨いて
ぐ - いで              急ぐ - 急いで
う、つ、る - って       洗う - 洗って    待つ - 待って    やる - やって
ぶ、む、ぬ - んで       遊ぶ - 遊んで    読む - 読んで    死ぬ - 死んで
特例：行く - 行って

ない形：う段 - あ段（特例：う - わ） + ない

た形：把“て”形的“て”换成“た”，把“で”换成“だ”

命令形：う段 - え段

意志形：う段 - お段 + う



二类动词（上一段动词、下一段动词）
い段+る    上一段动词
え段+る    下一段动词

ます形：る - ます

て形：る - て

ない形：去掉る + ない

た形：把“て”形的“て”换成“た”

命令形：る - ろ

意志形：る - よう



三类动词（カ变、サ变）
くる
する

ます形：くる - きます    する - します

て形：くる - きて    する - して

ない形：くる - こない    する - しない

た形：把“て”形的“て”换成“た”

命令形：くる - こい    する - しろ

意志形：くる - こよう    する - しよう
```
## 形容词
```
一类形：
い结尾

肯定：い + です
否定：い - くない + です（ないです和ありません可以相互替换）
过去式：い - かった + です
过去否定：い - くなかった + です
过去否定：い - くありませんでした

一类形 + 名

て形：い - くて

表示性质或状态的变化：い - くなります
因主语的意志性动作、作用而引起事物发生变化的场合：い - くします
比较两个以上事物的性质，认为其中一个比较好时：一类形 + ほうが　いいです



二类形：
綺麗　きれい
詳細　しょうさい
愉快　ゆかい
好き　すき
嫌い　きらい
にぎやか
静か　しずか
便利　べんり
不便　ふべん
暇　　ひま
元気　げんき
簡単　かんたん
親切　しんせつ
有名　ゆうめい

肯定：です
否定：では　ありません
过去式：でした
过去否定：では　ありませんでした

二类形 + な + 名

て形：二类形 + で

表示性质或状态的变化：二类形 + に　なります
因主语的意志性动作、作用而引起事物发生变化的场合：二类形 + に　します
比较两个以上事物的性质，认为其中一个比较好时：二类形 + な　ほうが　いいです
```
