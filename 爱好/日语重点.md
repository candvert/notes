```
“何”的读法 —— “なん” 和 “なに”
   在助词“の”以及量词前读“なん”。在“で”前两种读法都可以。


三类动词“します”前接汉字词或外来词可以构成新的三类动词，如“掃除します”。以这种形式构成的三类动词也可以说成“名词+を+します”，如“勉強を　します”。


“今日”　“明日”　“今年”后续“～中に”时，“～中に”读“～じゅうに”；“～週”　“～月”后续“～中に”时，“～中に”读“～ちゅうに”。


人和动物发出的声音为“声”，物体发出的声音为“音”。


日语的“趣味”是“爱好” “喜好”的意思。表示“兴趣”的时候，使用“興味”一词。


在表示不定数量时，日语中疑问词后面必须加“か”。
   仕事の後で，何度か　見に　行った　ことが　あります。


“～ちゃった”是第33课学过的“～てしまった”的口语形式。
   見ちゃった。今，こっそりつまみ食いしたでしょう。


第31课学习了一类形容词词尾“い”变成“く”后可以用做副词，不过“遠く”　“近く”属例外。它们一般不用来修饰动词，而是用做名词。


日语中一部分动词的“ます形”去掉“ます”可以作名词使用。如休み、考え、手伝い、乗り換え、持ち帰り、申し出。


日语的动词用来表示某种性质时，需用“～ています”的形式，而不能只用“ます形”。
   デザインがしゃれていて，人気があるみたいですね。


“似ます”只能以这种“～ています”的形式使用。
   なんか東京に似ていますね。
```

## 动词
```
   日语的动词可分为自动词和他动词。自动词是不带宾语的动词，表示不考虑外力影响的情况下，主语自主地进行动作，或自然地发生变化，其主语用“が”表示。最典型的他动词具有主语作用于宾语而使其发生变化的含义，其宾语用“を”表示。


动词的“基本形”　“ない形”　“た形”都属于简体形。


て形表示动作相继发生、正在做某事、请求某人做某事、许可、禁止、并列、动作结束后留下的结果状态、反复、习惯


一类动词（五段动词）
1. 结尾是う段非る的动词，肯定是五段动词
2. あ段+る、う段+る、お段+る
3. 特例 切る/きる　帰る/かえる　走る/はしる

ます形：う段 - い段 + ます

て形：
す - して              話す - 話して
く - いて              磨く - 磨いて
ぐ - いで              急ぐ - 急いで
う、つ、る - って       洗う - 洗って    待つ - 待って    やる - やって
ぶ、む、ぬ - んで       遊ぶ - 遊んで    読む - 読んで    死ぬ - 死んで
特例：行く - 行って

ない形：う段 - あ段（特例：う - わ） + ない

た形：把“て”形的“て”换成“た”，把“で”换成“だ”

命令形：う段 - え段

意志形：う段 - お段 + う

ば形：う段 - え段 + ば

可能形式：う段 - え段 + る

被动形式：う段 - あ段（特例：う - わ） + れる

使役形式：う段 - あ段（特例：う - わ） + せる

使役被动形式：う段 - あ段（特例：う - わ） + される



二类动词（上一段动词、下一段动词）
い段+る    上一段动词
え段+る    下一段动词

ます形：る - ます

て形：る - て

ない形：る - ない

た形：把“て”形的“て”换成“た”

命令形：る - ろ

意志形：る - よう

ば形：る - れば

可能形式：る - られる

被动形式：る - られる

使役形式：る - させる

使役被动形式：る - させられる



三类动词（カ变、サ变）
くる
する

ます形：くる - きます    する - します

て形：くる - きて    する - して

ない形：くる - こない    する - しない

た形：把“て”形的“て”换成“た”

命令形：くる - こい    する - しろ

意志形：くる - こよう    する - しよう

ば形：くる - くれば    する - すれば

可能形式：くる - こられる    する - できる

被动形式：くる - こられる    する - される

使役形式：くる - こさせる    する - させる

使役被动形式：くる - こさせられる
```

## 形容词
```
形容词和名词的て形表示并列


一类形：
い结尾
```
