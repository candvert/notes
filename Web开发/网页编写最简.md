```
1. 使用python将当前目录变成服务器目录
python -m http.server 8000
2. 访问http:\\localhost:8000
3. 编写html，css，javascript。需要注意typescript无法直接运行，需要转译为javascript。所以在html的script标签中只能引入javascript文件
<!DOCTYPE html>
<html lang="cn">
 <head>
    <meta charset="UTF-8">
    <link rel="stylesheet" href="a.css">
    <script type="module" src="a.js"></script>
    <title>Document</title>
</head>
<body>
    <p>hello</p>
</body>   
</html>
```
