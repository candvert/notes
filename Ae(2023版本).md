## 快捷键
```
Ctrl + N                                    新建合成
Ctrl + I                                    导入素材
V                                           选取
按住空格或中键                               移动
滚轮                                        缩放


F                                           蒙版羽化
Alt + 滚轮                                  缩放时间轴
B                                           开始
N                                           结束
Page Up                                     向前移动一帧
Page Down                                   向后移动一帧
```
